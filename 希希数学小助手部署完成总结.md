## 🏗️ 技术架构

### 前端小程序
- **框架**: 微信小程序原生开发
- **页面**: 首页、拍照、学习对话、结果展示、个人资料
- **功能**: 拍照识题、AI对话、学习报告、历史记录

### 后端云函数
- **平台**: 微信云开发
- **运行时**: Node.js 16.13
- **API集成**: 通义千问AI、微信OCR
- **数据库**: 云开发数据库

### AI能力
- **OCR识别**: 微信云OCR API
- **AI对话**: 通义千问API (苏格拉底式教学)
- **智能分析**: 题目分析、学习评估、报告生成
